The SWISH chat services allows you to chat with users about a file. SWISH displays an avatar for each user with whom you share an open file. If you select File/Chat ... the current notebook or program gets a chat window displayed below it, preloaded with older chat messages about this file.

The Send button provides additional options such as sending your current query, which appears as a button in the chat.

Status of the SWISH sharing service

What you are seeing are the first steps to allow users of SWISH to communicate. There are many facilities we have in mind, such as:

  • Connect editors for collaborative editing
  • Make teams that connect people also if they are not editing the same file.
  • Allow Prolog programs to exchange messages through the chat room.
